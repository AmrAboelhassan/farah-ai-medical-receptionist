# Farah – AI Medical Receptionist

Farah is an AI medical receptionist concept developed during my AI MicroMind Agent Builder internship.

The system simulates how a clinic receptionist can support new and returning patients through an AI-assisted workflow, including registration, verification, symptom collection, and doctor matching.

## Problem

Clinics often spend time handling repeated patient questions, registration steps, appointment requests, and basic routing. Farah explores how AI agents can support this workflow by automating repetitive front-desk interactions while keeping the process structured and trackable.

## Key Features

- New patient registration
- Returning patient verification
- Patient data collection
- Symptom handling
- Doctor matching based on patient needs
- PostgreSQL database integration
- Arabic and English interaction
- Webhook notifications for real-time updates

## Workflow

1. The patient starts a conversation with the AI receptionist.
2. The system identifies whether the patient is new or returning.
3. New patients are registered and assigned a patient ID.
4. Returning patients are verified using stored information.
5. The AI collects symptoms and helps route the patient to a suitable doctor.
6. A notification is sent through a webhook to support follow-up.
